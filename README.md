

# [会员管理系统](https://gitee.com/tuodagitee/membercms)

为线下商店提供会员管理系统，日常可以维护会员、商品信息，会员可以进行充值，也可以线上收银，支付时候如果选择会员卡可以进行扣费，系统提供了统计分析功能可以直观的了解目前会员及营业额
